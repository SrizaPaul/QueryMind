# QueryMind
AI-powered Natural Language SQL Analytics Dashboard with CSV upload, authentication, dynamic charts, query history, and PostgreSQL integration.

Features :

1.User Authentication (JWT)
2.Secure Login & Signup
3.CSV File Upload
4.Automatic PostgreSQL Table Creation
5.Natural Language Query Processing
6.SQL Query Generation
7.Query History Tracking
8.Interactive Dashboard
8.Bar Chart Visualization
9.Pie Chart Analytics
10.Line Graph Trends
11.CSV Export
12.Protected Routes
13.Responsive UI

Tech Stack:

Frontend: 

-React.js
-Tailwind CSS
-Axios
-React Router DOM
-Recharts
-React CSV

Backend:

-Node.js
-Express.js
-PostgreSQL
-JWT Authentication
-Multer
-CSV Parser
-bcryptjs

Database:

-PostgreSQL
